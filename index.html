Setup guide ▼

ⓘ The ZIP export works from any browser with no setup needed. The Cloudflare Worker is only needed for the Edit Existing Records operation.
1
Go to workers.cloudflare.com and sign up for a free account.
2
Click Create Worker and delete all the default code in the editor.
3
Download and open kobo_cors_proxy_worker.js, copy all its contents, and paste into the Cloudflare editor.
4
Click Save & Deploy. Copy your permanent Worker URL:
https://kobo-proxy.YOUR-NAME.workers.dev
5
In the Connection section, set Server to Custom URL…, paste your Worker URL, and click Test connection.
ⓘ The Worker only forwards requests to eu.kobotoolbox.org. Your API token is never stored or logged.
📃 Session-based. Run a local server — must repeat each session.
1
Make sure Python is installed. Open a terminal (Windows: Win+R → cmd).
2
Navigate to the folder where this file is saved:
cd C:\Users\YourName\Downloads
3
Start the local server:
python -m http.server 8080
4
Open Chrome and go to:
http://localhost:8080/kobo_tool.html
ⓘ Replace the filename if you renamed this file.
5
Keep the terminal open while using the tool. Next time, repeat steps 3–4 only.
6
Still blocked on localhost? Run a CORS proxy in a second terminal:
npx local-cors-proxy --proxyUrl https://eu.kobotoolbox.org --port 8010
Then set the server URL to http://localhost:8010. Requires Node.js.
ⓘ Click any command above to copy it to your clipboard.
